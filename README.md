# DJ Music System - Flask

Sistema de música para DJ desenvolvido com Python, Flask, JWT e Swagger.

## Descrição

Este projeto foi desenvolvido para a Atividade Prática II. A aplicação possui cadastro de usuários, login com geração de token JWT, rota protegida para listagem de músicas, interface HTML integrada à API e documentação Swagger.

## Tecnologias utilizadas

- Python
- Flask
- Flask-JWT-Extended
- Flask-CORS
- Flasgger
- Swagger
- HTML
- JavaScript

## Funcionalidades

- Cadastro de usuários
- Login de usuários
- Geração de token JWT
- Proteção de rota com JWT
- Listagem de músicas
- Interface web simples
- Documentação da API com Swagger
